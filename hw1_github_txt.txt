TXT 
1. Создать внешний репозиторий c названием TXT.          /Repositories/New/Create a new repository/Repository name/TXT/*Public/Add a README file/Create repository
2. Клонировать репозиторий TXT на локальный компьютер.   $ git clone https://github.com/shyri4ek/TXT
3. Внутри локального TXT создать файл “new.txt”.         $ cd TXT $ touch new.txt
4. Добавить файл под гит.                                $ git status $ git add .
5. Закоммитить файл.                                     $ git commit -m "add new.txt file"
6. Отправить файл на внешний GitHub репозиторий.         $ git push
7. Отредактировать содержание файла “new.txt” - написать информацию о себе  $ vim new.txt/ i / вводим текст 
(ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Попова Александра Владимировна
Всё написать в формате TXT.                                              37 лет
                                                                         1
                                                                         2132 евро в месяц
                                                                        / Esc /:wq
8. Отправить изменения на внешний репозиторий. $ git add new.txt 
                                               $ git commit -m "add text to new.txt" 
                                               $ git push
9. Создать файл preferences.txt                $ touch preferences.txt

10. В файл preferences.txt” добавить информацию о своих предпочтениях  $ vim preferences.txt / i / вводим текст 
(Любимый фильм, любимый сериал, любимая еда, любимое время года,       Дневник памяти
стрaна которую хотели бы посетить) в формате TXT.                      Секс в большом городе
                                                                       Фрукты/овощи
                                                                       Лето
                                                                       Индия
                                                                       / Esc /:wq

11. Создать файл skills.txt добавить информацию о скиллах  $ touch skills.txt  
    которые будут изучены на курсе в формате TXT           $ vim skills.txt/ i / вводим текст / Esc /:wq

12. Сделать коммит в одну строку. $ git add . 
                                  $ git commit -m "add text to preferences.txt and skills.txt"
 
13. Отправить сразу 2 файла на внешний репозиторий. $ git push

14. На веб интерфейсе создать файл bug_report.txt. / Add file / Create new file / Name your file... / bug_report.txt /Commit new file
15. Сделать Commit changes (сохранить) изменения на веб интерфейсе. /в строке Commit changes пишем новый Commit                                                                   

16. На веб интерфейсе модифицировать файл bug_report.txt,         / bug_report.txt / Edit file / вводим текст
добавить баг репорт в формате TXT. 
                                                                  1.Название: Не работает кнопка отправки заказа.
                                                                  2.Как воспроизвести?
                                                                  2.1. Открыть сайт *.
                                                                  2.2. Нажать по ссылке *.
                                                                  2.3. Пролистать до поля *.
                                                                  2.4. Ввести значение в поле *.
                                                                  2.5. Попытаться нажать кнопку отправки заказа.
                                                                  3.Фактический результат: кнопка неактивна.
                                                                  4.Ожидаемый результат: кнопка нажимается, можно сделать заказ.
                                                                  5.Вложения
                                                                  / Commit changes

17. Сделать Commit changes (сохранить) изменения на веб интерфейсе. /в строке Commit changes пишем новый Commit 

18. Синхронизировать внешний и локальный репозиторий TXT  $ git pull
